# Основы ООП в PHP
Объектно-ориентированное программирование (ООП) в PHP позволяет создавать более структурированные и масштабируемые приложения. В основе ООП лежат классы и объекты. Давайте разберем основные понятия и примеры, как это работает в PHP.

## 1. Что такое класс?
Класс — это шаблон, из которого создаются объекты. Класс описывает свойства (данные) и методы (функции), которые могут быть использованы для взаимодействия с этими свойствами.

## 2. Что такое объект?
Объект — это экземпляр класса. Это конкретная реализация класса с собственными значениями для свойств.

## 3. Свойства и методы
- Свойства — это переменные, которые хранят данные, связанные с классом.
- Методы — это функции, которые выполняют действия с этими данными.

## 4. Структура класса и объекта в PHP
Пример простого класса:
```php
<?php
// Класс "Car" (автомобиль)
class Car {
    // Свойства класса
    public $color;
    public $model;

    // Конструктор (специальный метод для инициализации объектов)
    public function __construct($color, $model) {
        $this->color = $color;
        $this->model = $model;
    }

    // Метод класса
    public function displayInfo() {
        echo "Модель автомобиля: " . $this->model . "<br>";
        echo "Цвет автомобиля: " . $this->color . "<br>";
    }
}

// Создание объекта (экземпляра класса)
$car1 = new Car("Красный", "BMW");
$car2 = new Car("Синий", "Mercedes");

// Вызов метода объекта
$car1->displayInfo();
$car2->displayInfo();
?>
```

## 5. Файл и класс
- В PHP обычно рекомендуется, чтобы имя файла совпадало с именем класса, а файл должен содержать только один класс.
- Пример: файл называется `Car.php`, а в нем класс `Car`.

## 6. Именование классов
- В PHP принято использовать стиль camelCase для именования классов.
- Первый символ каждого слова в классе пишется с заглавной буквы, начиная с первой буквы. Например: `CarModel`, `EmployeeDetails`.

## 7. Подключение классов
Чтобы использовать классы в разных файлах, можно подключать их с помощью `require`, `include`, или использовать автозагрузку (например, через Composer).

Пример подключения класса:
```php
// Подключаем файл с классом Car
require_once 'Car.php';

// Создаем объект класса
$car1 = new Car("Зеленый", "Audi");
$car1->displayInfo();
```

# Переменные и константы
В PHP переменные и константы имеют свои особенности в создании и использовании. Вот основные правила для каждой из них:

## 1. `Переменные в PHP`:
- `Регистрозависимость`: Да, переменные в PHP регистрозависимы. То есть $variable и $Variable — это разные переменные.
- `Правила именования переменных`:
  * Переменная должна начинаться с символа доллара (`$`), за ним идет имя переменной.
  * Имя переменной должно начинаться с `буквы` (A-Z, a-z) или символа подчеркивания (_), а затем могут следовать буквы, цифры (0-9) и подчеркивания.
  * Переменная не может начинаться с цифры. Например, `$1variable` — это недопустимое имя переменной.
- `Запрещенные символы`:
  * В имени переменной нельзя использовать пробелы или специальные символы, такие как `@`, `#`, `$` и т. д. Разрешены только буквы, цифры и подчеркивания.
- `Нельзя использовать зарезервированные имена`: Имя переменной не может совпадать с зарезервированными словами или ключевыми словами PHP, такими как `if`, `while`, `function`, `class` и другие.

Пример корректных переменных:
```php
$variable;
$_variable;
$var123;
```

Пример некорректных переменных:
```php
$1variable;  // начинается с цифры
$my-variable;  // содержит дефис
$if;  // зарезервированное слово
```

## 2. `Константы в PHP`:
- `Константы регистрозависимы`: По умолчанию имена констант в PHP не регистрозависимы. Однако, начиная с PHP 7, можно использовать флаг `define()` для создания регистрозависимых констант (по умолчанию они создаются без учета регистра).
- `Правила именования`:
  * Константы обычно пишутся в `верхнем регистре` (по соглашению, хотя это не обязательное правило, но принято для лучшей читаемости).
  * Имя константы должно состоять из букв, цифр и символа подчеркивания (`_`).
  * Константы `не могут начинаться с цифры`.
- `Зарезервированные имена`: В случае с константами, важно также избегать использования зарезервированных имен, хотя зарезервированные слова PHP не могут быть использованы как имена констант.

Пример создания и использования констант:
```php
define('MY_CONSTANT', 'value');
echo MY_CONSTANT;  // Выведет 'value'
```

### Итог:
- Переменные в PHP регистрозависимы, не могут начинаться с цифры, могут начинаться с символа подчеркивания и не могут использовать зарезервированные имена.
- Константы по умолчанию не регистрозависимы и также не могут начинаться с цифры, но обычно пишутся в верхнем регистре для удобства.

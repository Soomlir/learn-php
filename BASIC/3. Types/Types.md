# Типы данных в PHP
В PHP существует девять основных типов данных. 

## 1. Целые числа (int или integer)
Целые числа — это числа без десятичной точки. В PHP они могут быть как положительными, так и отрицательными.
```php
$int = 42;
```

## 2. Числа с плавающей точкой (float или double)
Числа с плавающей точкой — это числа, содержащие дробную часть. Такие числа могут быть записаны с точкой или через экспоненциальную форму.
```php
$float = 3.14;
$scientific = 1.23e4; // 1.23 * 10^4
```

## 3. Строки (string)
Строки — это последовательности символов, заключенные в одиночные или двойные кавычки.
```php
$string = "Hello, world!";
```

## 4. Булевы значения (bool)
Булевы значения могут быть только двумя возможными: true или false. Эти значения обычно используются в условных выражениях.
```php
$bool = true;
```

## 5. Массивы (array)
Массивы в PHP могут содержать несколько значений, которые могут быть разных типов. Массивы бывают индексированные и ассоциативные.
```php
$array = [1, 2, 3];
$assocArray = ['name' => 'John', 'age' => 25];
```

## 6. Объекты (object)
Объекты — это экземпляры классов. Они могут содержать свойства и методы.
```php
class Person {
    public $name;
    public function __construct($name) {
        $this->name = $name;
    }
}

$person = new Person("John");
```

## 7. NULL
NULL — это специальное значение, которое означает отсутствие значения или переменной. Переменная, содержащая `NULL`, не имеет значения.
```php
$var = NULL;
```

## 8. Ресурсы (resource)
Ресурсы — это специальные переменные, которые хранят ссылки на внешние ресурсы, такие как соединения с базой данных, файлы, изображения и т. п. Ресурсы обычно создаются и управляются с помощью специфичных для PHP функций.
```php
$file = fopen("file.txt", "r");
```

## 9. Callable (колбэки)
Тип callable в PHP обозначает возможность передачи функции или метода как параметра в другую функцию. В PHP функции и методы могут быть вызваны с помощью переменных.
```php
function greet($name) {
    echo "Hello, $name!";
}

$callback = "greet";
$callback("John");
```

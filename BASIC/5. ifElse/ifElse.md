# Условные операторы в PHP
## if
Условный оператор if выполняет блок кода, если условие истинно.
```php
if ($x > 10) {
    echo "x больше 10";
}
```

## if-else
Если условие ложно, выполняется блок кода в else.
```php
if ($x > 10) {
    echo "x больше 10";
} else {
    echo "x меньше или равно 10";
}
```

## if-elseif-else
Позволяет проверять несколько условий.
```php
if ($x > 10) {
    echo "x больше 10";
} elseif ($x == 10) {
    echo "x равно 10";
} else {
    echo "x меньше 10";
}
```

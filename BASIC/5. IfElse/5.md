# Условные ветления в PHP
В PHP конструкция `else if` используется для выполнения различных блоков кода в зависимости от результата нескольких условий. Она является расширением оператора `if` и позволяет проверить несколько условий поочередно.

Вот как работает структура `if`, `else if` и `else`:
## Основная структура:
```php
if (условие1) {
    // Выполняется, если условие1 истинно
} elseif (условие2) {
    // Выполняется, если условие1 ложно, но условие2 истинно
} else {
    // Выполняется, если ни одно из условий не истинно
}
```

## Пример:
```php
<?php
$age = 20;

if ($age < 18) {
    echo "Вы слишком молоды, чтобы войти.";
} elseif ($age >= 18 && $age < 21) {
    echo "Вы совершеннолетний, но еще не можете пить алкоголь.";
} else {
    echo "Вы взрослый, можете делать все, что хотите.";
}
?>
```

## Объяснение:
1. Сначала проверяется условие в `if`. Если оно истинно, то выполняется его блок.
2. Если условие в `if` ложно, то проверяется условие в `elseif`. Если оно истинно, выполняется его блок.
3. Если все условия в `if` и `elseif` ложны, выполняется блок `else` (если он есть).

## Разница между `else if` и `elseif`:
- В PHP можно использовать как `elseif`, так и `else if` (с пробелом). Оба варианта работают одинаково, но принято использовать `elseif` без пробела, так как это более компактный и читаемый вариант.

## Пример с `else if`:
```php
<?php
$age = 16;

if ($age < 18) {
    echo "Меньше 18 лет.";
} else if ($age == 18) {
    echo "Тебе 18 лет!";
} else {
    echo "Ты старше 18 лет.";
}
?>
```

Обе формы (`elseif` и `else if`) могут быть использованы, но для чистоты кода и соответствия стандартам кодирования рекомендуется использовать `elseif`.

# Основы PHP
PHP — это популярный серверный язык программирования, используемый для создания динамических веб-страниц и веб-приложений. Вот основные аспекты синтаксиса PHP:

## Старт и завершение PHP-кода
PHP-код пишется внутри специальных тегов:
```php
<?php
  // Ваш PHP код
?>
```
Это позволяет серверу интерпретировать содержимое как код PHP, а не как обычный HTML.

## Комментарии
В PHP поддерживаются несколько типов комментариев:

Однострочные комментарии:
```php
// Это однострочный комментарий
# Это тоже однострочный комментарий
```

Многострочные комментарии:
```php
/* Это
   многострочный
   комментарий
*/
```

## Строки
В PHP разница между одинарными (`'`) и двойными (`"`) кавычками заключается в том, как они обрабатывают строковые данные.

### 1. Одинарные кавычки ('):
- Строки в одинарных кавычках воспринимаются как буквальные строки.
- Переменные не интерпретируются внутри строки. То есть, если вы вставите переменную в строку, она будет просто выведена как текст.
- Специальные символы обрабатываются ограниченно:
  * Экранированный апостроф (`\'`) и экранированный обратный слэш (`\\`) будут распознаны.
```php
$name = 'Иван';
echo 'Привет, $name!'; // Выведет: Привет, $name!
```

### 2. Двойные кавычки ("):
- Строки в двойных кавычках могут интерпретировать переменные и специальные символы.
- Переменные внутри строки будут заменены на их значения.
- Поддерживаются специальные символы, такие как:
  * \n — новая строка
  * \t — табуляция
  * \$ — долларовый знак
```php
$name = 'Ольга';
echo "Привет, $name!"; // Выведет: Привет, Ольга!
```

## Итог:
- Одинарные кавычки: быстрее и проще, так как не требуется обработка переменных и спецсимволов, за исключением экранирования апострофа и обратного слэша.
- Двойные кавычки: более гибкие, так как поддерживают интерполяцию переменных и спецсимволов.

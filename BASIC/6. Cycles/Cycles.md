# Циклы в PHP

## Цикл while
Цикл while выполняет блок кода, пока выполняется заданное условие. 
```php
while (условие) {
    // код, который будет выполняться, пока условие истинно
}
```

```php
$i = 0;
while ($i < 5) {
    echo "Итерация $i<br>";
    $i++;
}
```

```php
Итерация 0
Итерация 1
Итерация 2
Итерация 3
Итерация 4
```

## Цикл do-while
Цикл do-while сначала выполняет код, а затем проверяет условие. Это гарантирует, что код выполнится хотя бы один раз. 
```php
do {
    // код, который будет выполняться хотя бы один раз
} while (условие);
```

```php
$i = 0;
do {
    echo "Итерация $i<br>";
    $i++;
} while ($i < 5);
```

```php
Итерация 0
Итерация 1
Итерация 2
Итерация 3
Итерация 4
```

## Цикл for
Цикл for используется, когда известно количество итераций заранее. 
```php
for (инициализация; условие; шаг) {
    // код, который будет выполняться на каждой итерации
}
```

```php
for ($i = 0; $i < 5; $i++) {
    echo "Итерация $i<br>";
}
```

```php
Итерация 0
Итерация 1
Итерация 2
Итерация 3
Итерация 4
```

## Цикл foreach
Цикл foreach используется для перебора элементов массива или объектов.
```php
foreach ($array as $value) {
    // код, который будет выполняться для каждого элемента массива
}
```

Если нужно получить и ключ, и значение:
```php
foreach ($array as $key => $value) {
    // код, который будет выполняться для каждого элемента массива
}
```

Пример:
```php
$array = [1, 2, 3, 4, 5];
foreach ($array as $value) {
    echo "Значение: $value<br>";
}
```

Этот цикл выведет:
```php
Значение: 1
Значение: 2
Значение: 3
Значение: 4
Значение: 5
```

Если нужно получить ключи и значения:
```php
$array = [1 => 'a', 2 => 'b', 3 => 'c'];
foreach ($array as $key => $value) {
    echo "Ключ: $key, Значение: $value<br>";
}
```

Этот цикл выведет:
```php
Ключ: 1, Значение: a
Ключ: 2, Значение: b
Ключ: 3, Значение: c
```

Примечания
- continue: Останавливает текущую итерацию цикла и продолжает выполнение с следующей.
- break: Прерывает выполнение цикла полностью.

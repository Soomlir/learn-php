# Загрузка файлов в PHP
Загрузка файлов в PHP — это процесс, при котором пользователь отправляет файл с его устройства на сервер через веб-форму. PHP предоставляет несколько функций для обработки загруженных файлов, включая проверку, перемещение файлов и работу с их метаданными.

## Как работает загрузка файлов в PHP
1. `HTML форма для загрузки файлов` Для начала необходимо создать HTML форму, в которой будет возможность выбрать файл для загрузки.
```html
<form action="upload.php" method="post" enctype="multipart/form-data">
    <label for="fileToUpload">Выберите файл для загрузки:</label>
    <input type="file" name="fileToUpload" id="fileToUpload">
    <input type="submit" value="Загрузить файл" name="submit">
</form>
```
- Атрибут `enctype="multipart/form-data"` необходим для передачи файлов в HTTP-запросе.
- В элементе `<input type="file">` пользователь выбирает файл для загрузки.

2. `PHP обработчик загрузки файла` На серверной стороне файл будет обрабатываться через суперглобальный массив `$_FILES`. Этот массив содержит информацию о загруженном файле, такую как имя, тип, временный путь и размер.

### Основные элементы массива `$_FILES`:
- `$_FILES['fileToUpload']['name']` — имя файла.
- `$_FILES['fileToUpload']['type']` — MIME-тип файла (например, "image/jpeg").
- `$_FILES['fileToUpload']['tmp_name']` — временный путь, по которому PHP сохраняет загруженный файл на сервере.
- `$_FILES['fileToUpload']['error']` — код ошибки, если она произошла.
- `$_FILES['fileToUpload']['size']` — размер файла в байтах.

## Пример кода для обработки загрузки файла:
```php
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    // Проверяем, был ли загружен файл
    if (isset($_FILES["fileToUpload"])) {
        $target_dir = "uploads/"; // Папка для загрузки
        $target_file = $target_dir . basename($_FILES["fileToUpload"]["name"]);
        $uploadOk = 1;
        $imageFileType = strtolower(pathinfo($target_file, PATHINFO_EXTENSION));

        // Проверяем, является ли файл изображением
        if (isset($_POST["submit"])) {
            $check = getimagesize($_FILES["fileToUpload"]["tmp_name"]);
            if ($check !== false) {
                echo "Файл — это изображение. Тип: " . $check["mime"] . ".<br>";
                $uploadOk = 1;
            } else {
                echo "Файл не является изображением.<br>";
                $uploadOk = 0;
            }
        }

        // Проверяем размер файла
        if ($_FILES["fileToUpload"]["size"] > 500000) { // Максимальный размер 500 КБ
            echo "Извините, файл слишком большой.<br>";
            $uploadOk = 0;
        }

        // Разрешаем только определенные форматы файлов
        if ($imageFileType != "jpg" && $imageFileType != "png" && $imageFileType != "jpeg" && $imageFileType != "gif") {
            echo "Извините, только файлы JPG, JPEG, PNG и GIF разрешены.<br>";
            $uploadOk = 0;
        }

        // Проверяем, был ли выявлен какой-либо ошибка при загрузке
        if ($_FILES["fileToUpload"]["error"] != 0) {
            echo "Ошибка при загрузке файла: " . $_FILES["fileToUpload"]["error"] . "<br>";
            $uploadOk = 0;
        }

        // Если все проверки пройдены, перемещаем файл в целевую папку
        if ($uploadOk == 1) {
            if (move_uploaded_file($_FILES["fileToUpload"]["tmp_name"], $target_file)) {
                echo "Файл ". htmlspecialchars(basename($_FILES["fileToUpload"]["name"])) . " был загружен.<br>";
            } else {
                echo "Извините, произошла ошибка при загрузке файла.<br>";
            }
        }
    }
}
?>
```

## Описание кода:
1. `Проверка метода запроса`: `$_SERVER["REQUEST_METHOD"] == "POST"` — мы проверяем, что запрос пришел методом POST, так как загрузка файла происходит именно через этот метод.
2. `Проверка ошибок`: Проверяем, был ли загружен файл с помощью `$_FILES['fileToUpload']['error']`. Код ошибки можно сравнивать с константами, например, `UPLOAD_ERR_OK` для успешной загрузки.
3. Проверка типа файла: В коде проверяется тип файла с помощью `getimagesize()` для изображений. Вы можете заменить это на проверку других типов файлов, если нужно.
4. `Проверка размера файла`: Мы ограничиваем размер файла (например, 500 КБ) с помощью `$_FILES['fileToUpload']['size']`.
5. `Перемещение файла`: Если все проверки прошли успешно, файл перемещается с временного места в целевую папку с помощью `move_uploaded_file()`.

## Функции обработки файлов в PHP
1. `move_uploaded_file()` — перемещает загруженный файл из временного места в целевую папку. Это обязательная функция для работы с загруженными файлами.
```php
move_uploaded_file($_FILES["file"]["tmp_name"], "uploads/" . $_FILES["file"]["name"]);
```
2. `getimagesize()` — используется для проверки, является ли файл изображением, а также для получения его размеров и MIME-типов.
```php
$imageInfo = getimagesize($_FILES["fileToUpload"]["tmp_name"]);
echo "Ширина: " . $imageInfo[0] . ", Высота: " . $imageInfo[1] . ", MIME: " . $imageInfo["mime"];
```
3. `file_exists()` — проверяет, существует ли файл в целевой директории.
```php
if (file_exists($target_file)) {
    echo "Извините, файл уже существует.";
}
```
4. `is_uploaded_file()` — проверяет, был ли файл загружен с помощью HTTP POST.
```php
if (is_uploaded_file($_FILES["fileToUpload"]["tmp_name"])) {
    echo "Файл был загружен.";
}
```
5. `move_uploaded_file()` — перемещает файл в нужную директорию. Очень важно использовать эту функцию, чтобы предотвратить возможные угрозы безопасности.
```php
move_uploaded_file($_FILES['fileToUpload']['tmp_name'], $target_file);
```

## Примечания по безопасности
- `Проверка типа файлов`: Чтобы избежать загрузки опасных файлов (например, исполнимых), важно проверять MIME-тип и расширение файла.
- `Ограничение размера`: Убедитесь, что размер файлов ограничен, чтобы избежать чрезмерного использования серверного пространства.
- `Использование уникальных имен файлов`: Используйте уникальные имена для файлов, чтобы избежать перезаписи существующих файлов на сервере. Это можно сделать, например, добавляя метки времени или уникальные идентификаторы.

Пример с уникальными именами:
```php
$target_file = $target_dir . uniqid() . "-" . basename($_FILES["fileToUpload"]["name"]);
```

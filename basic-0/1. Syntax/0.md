# Основы синтаксиса языка PHP
PHP (Hypertext Preprocessor) — это серверный язык программирования, который в основном используется для создания веб-приложений. Синтаксис PHP похож на другие языки программирования, такие как C, C++, JavaScript и Java. Вот основные моменты синтаксиса:

## 1. `Теги PHP`: 
PHP код обычно пишется между тегами <?php и ?>. Внутри этих тегов PHP выполняет код.
```php
<?php
   echo "Hello, World!";  // Выводит "Hello, World!"
?>
```

## 2. `Строки и вывод`: 
В PHP можно использовать строки как в одинарных, так и в двойных кавычках. В двойных кавычках можно использовать переменные, а в одинарных — нет.
```php
<?php
   $name = "John";
   echo "Hello, $name!";  // Выведет "Hello, John!"
   echo 'Hello, $name!';  // Выведет "Hello, $name!"
?>
```

## 3. `Переменные`: 
Переменные в PHP начинаются с символа $, за которым идет имя переменной.
```php
<?php
   $age = 25;
   $city = "Moscow";
   echo $age;  // Выведет 25
   echo $city; // Выведет Moscow
?>
```

## 4. Закрывающие и открывающие теги:
В PHP рекомендуется всегда закрывать теги ?>, но в некоторых случаях они могут быть опущены, если код заканчивается на PHP (это помогает избежать ошибок с пробелами в конце файла).

## PHP показывает результат работы, а не сам код
PHP — это серверный язык программирования, который генерирует HTML или другие типы контента для веб-браузера. Когда сервер выполняет PHP-скрипт, он обрабатывает PHP-код и отправляет результат выполнения в браузер. Это значит, что сам код PHP не виден пользователю. На выходе браузер получает только результат работы скрипта (например, HTML).

Пример:
```php
<?php
   echo "Hello, World!";
?>
```

Когда этот код выполняется на сервере, браузер получит только:
```php
Hello, World!
```

Весь PHP код скрыт от конечного пользователя.

## Комментарии в PHP
В PHP существуют два типа комментариев:
1. `Однострочные комментарии`: Используются для комментариев, которые занимают одну строку. Это можно сделать с помощью `//` или `#`.
```php
<?php
   // Это однострочный комментарий
   # Это тоже однострочный комментарий
?>
```

2. `Многострочные комментарии`: Многострочные комментарии начинаются с `/*` и заканчиваются на `*/`. Этот тип используется для комментариев, которые могут занимать несколько строк.
```php
<?php
   /* Это многострочный
      комментарий, который занимает
      несколько строк. */
?>
```

Комментарии полезны для добавления пояснений в код и улучшения его читаемости, а также для временного исключения части кода из выполнения.

# require и include в PHP
В PHP `require` и `include` — это две конструкции, которые используются для включения внешних файлов в скрипт. Однако есть несколько важных различий между ними:

## 1. `require`:
- Используется для включения файла, и если файл не удаётся найти или подключить, PHP вызывает фатальную ошибку и прекращает выполнение скрипта.
- Это значит, что файл обязателен для корректной работы скрипта.
- Пример:
```php
require 'config.php';
```
- Если `config.php` не существует или его нельзя открыть, PHP завершит выполнение с ошибкой.

## 2. `include`:
- Включает файл, но если файл не удаётся подключить, PHP вызывает предупреждение (warning), а выполнение скрипта продолжается.
- Это позволяет не останавливать выполнение программы, даже если файл не найден.
- Пример:
```php
include 'header.php';
```
- Если `header.php` не существует или не может быть открыт, PHP выдаст предупреждение, но продолжит выполнение кода.


## Различия:
- Ошибка vs Предупреждение:
  * `require` вызывает фатальную ошибку, если файл не найден, а `include` — только предупреждение.
- Необходимость файла:
  * `require` применяется для обязательных файлов, без которых скрипт не сможет работать (например, для подключения к базе данных или конфигурационных файлов).
  * `include` используется для необязательных файлов (например, шаблонов или вспомогательных функций), которые могут быть загружены по желанию.

## Пример:
```php
// Пример с require
require 'config.php';  // Если файл не найден, выполнение скрипта будет остановлено

// Пример с include
include 'footer.php';  // Если файл не найден, выполнение продолжится, но будет выведено предупреждение
```

## Разница между `require_once` и `include_once`:
Обе эти конструкции работают так же, как `require` и `include`, но добавляют проверку на то, был ли файл уже подключён. Это предотвращает многократное подключение одного и того же файла.
- `require_once` и `include_once` подключают файл только один раз за выполнение скрипта, даже если их вызывают несколько раз.
```php
require_once 'config.php';
include_once 'header.php';
```

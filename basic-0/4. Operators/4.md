# Операторы в PHP
В языке программирования PHP существует множество операторов, которые выполняют различные операции над переменными и значениями. Операторы можно разделить на несколько типов в зависимости от их назначения. Давай рассмотрим их подробно.

## 1. `Арифметические операторы`
Эти операторы используются для выполнения математических операций.
- `+` — сложение.
- `-` — вычитание.
- `*` — умножение.
- `/` — деление.
- `%` — остаток от деления (модуль).
- `**` — возведение в степень (PHP 5.6+).

Примеры:
```php
$a = 5;
$b = 3;
echo $a + $b; // 8
echo $a % $b; // 2 (остаток от деления 5 на 3)
```

## 2. `Операторы сравнения`
Эти операторы используются для сравнения двух значений. Результат всегда будет `true` или `false`.
- `==` — равно (сравнение по значению).
- `===` — идентично (сравнение по значению и типу).
- `!=` — не равно (по значению).
- `!==` — не идентично (по значению и типу).
- `>` — больше.
- `<` — меньше.
- `>=` — больше или равно.
- `<=` — меньше или равно.

Примеры:
```php
$a = 5;
$b = '5';
var_dump($a == $b);  // true, так как значения одинаковые
var_dump($a === $b); // false, так как типы разные (int и string)
```

## 3. `Логические операторы`
Используются для выполнения логических операций. Чаще всего применяются в условных выражениях.
- `&&` — логическое И (оператор "и").
- `||` — логическое ИЛИ (оператор "или").
- `!` — логическое НЕ.
- `and` — логическое И (приоритет ниже, чем у `&&`).
- `or` — логическое ИЛИ (приоритет ниже, чем у `||`).

Примеры:
```php
$a = true;
$b = false;
var_dump($a && $b); // false
var_dump($a || $b); // true
var_dump(!$a);      // false
```

## 4. `Операторы присваивания`
Используются для присваивания значений переменным.
- `=` — обычное присваивание.
- `+=` — присваивание с добавлением.
- `-=` — присваивание с вычитанием.
- `*=` — присваивание с умножением.
- `/=` — присваивание с делением.
- `%=` — присваивание с остатком от деления.
- `.=` — присваивание строки (конкатенация).

Примеры:
```php
$a = 5;
$a += 3; // $a = 8
$a .= " years"; // $a = "8 years"
```

## 5. `Операторы инкремента и декремента`
Эти операторы используются для увеличения или уменьшения значения переменной на 1.
- `++` — инкремент (увеличение на 1).
- `--` — декремент (уменьшение на 1).
Они могут быть префиксными (до выражения) или постфиксными (после выражения):
```php
$a = 5;
echo ++$a; // 6 (инкремент до использования)
echo $a++; // 6 (инкремент после использования)
```

## 6. `Операторы работы с массивами`
Эти операторы используются для работы с массивами.
- `[]` — доступ к элементам массива.
- `array_key_exists()` — проверка существования ключа в массиве.
- `count()` — подсчёт элементов массива.
- `array_merge()` — объединение массивов.

Примеры:
```php
$array = ['apple', 'banana'];
echo $array[0]; // apple

$array2 = ['orange'];
$result = array_merge($array, $array2);
print_r($result); // Array ( [0] => apple [1] => banana [2] => orange )
```

## 7. `Операторы строк`
Операторы для работы со строками.
- `.` — оператор конкатенации (сцепление строк).
- `.=` — добавление строки к другой строке.

Пример:
```php
$a = "Hello";
$b = " world!";
$a .= $b;  // $a = "Hello world!"
```

## 8. `Операторы ошибок`
Используются для управления ошибками в коде.
- `@` — подавление ошибок (не рекомендуется для использования в крупных проектах).

Пример:
```php
$result = @file_get_contents('non_existent_file.txt');  // Ошибка будет подавлена
```

## 9. `Тернарный оператор`
Тернарный оператор — это короткая форма условного выражения.
- `condition ? expr1 : expr2` — если условие истинно, выполняется `expr1`, иначе — `expr2`.

Пример:
```php
$a = 5;
$b = ($a > 3) ? "Yes" : "No";  // $b = "Yes"
```

## 10. `Операторы типа`
Эти операторы используются для работы с типами данных.
- `is_*` — проверки типа данных, например, `is_int()`, `is_string()`, `is_array()`.
- `instanceof` — проверка типа объекта.

Пример:
```php
$a = 5;
var_dump(is_int($a));  // true
var_dump($a instanceof ClassName); // false (если $a не является объектом класса ClassName)
```
